Second Year Project

This project was created by myself and a classmate for 20% of our final grade in Year 2. It is a quick dungeon crawler with wave mechanics with the goal of seeing how long you can go.

Project Outcomes

Breath Search Algorithm

The breath search was a requirement for this projects. It acts off of the cell mapping in order to exapnds itself from point A to point B and find an efficient path that the enemies may then follow. This was out first experience in using a more difficult algorithm.

Cell based Mapping

The map is created using cells. These cells act as indicators for how the player interacts with the game. Cells manage position information, collision, weapon and enemy spawnings. They help keep an accurate log of what is happening during gameplay.

Thor Library Particles

Using SFML and the thor library. I learnt how to add, adjust and control particles for this game. Understanding particles helped me understand how to make the game feel more interactive and live rather than a flat 2D project.
